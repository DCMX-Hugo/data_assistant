# Data_assistant
一个简易的数据处理助手
# 功能介绍
## 1.端到端数据处理流程：

数据加载 → 清洗 → 分析 → 可视化 → 报告生成

自动处理CSV/Excel格式

## 2.智能数据清洗：

列名标准化（特殊字符处理）

缺失值处理（数值列用中位数，分类列用众数）

异常值检测与处理（IQR方法）

自动日期类型转换

## 3.全面数据分析：

描述性统计（数值+分类变量）

相关性分析

分类变量分布

时间序列分析

## 4.自动化可视化：

分布直方图

分类计数图

相关性热力图

变量关系矩阵

时间趋势图

## 5.专业报告生成：

内置响应式HTML模板

自动嵌入可视化图表（Base64编码）

详细记录数据处理步骤

错误日志跟踪

## 6.命令行接口：

支持自定义输出目录

友好状态提示（✅/❌）
# 使用方法
  ## 1.输入以下命令
    python -m data_assistant.main "yourdata.csv" -o reports
    运行完毕后会在reports（你选定的文件夹）内生成一个后缀为.html的文件
  ## 2.使用浏览器或编译器打开可以看到具体的数据清洗报告
